# PanelVault

PanelVault is an iPhone application for electrical contractors and distribution board manufacturers.

Goals:
- Archive every electrical project.
- Manage manufacturers.
- Manage customers.
- Store photos.
- Store PDFs.
- Search every project instantly.

Design Language:
- Dark theme.
- Purple accent.
- Rounded cards.
- Apple-inspired UI.
- Dashboard-first workflow.
